About the System

This blockchain-based organ donation management system implements the following algorithms:

  1. Algorithm 1: Patient Addition - Doctors add patients to the system
  2. Algorithm 2: Donor Addition - Doctors register donors
  3. Algorithm 3: Matching Process - Organizers match compatible patients and donors
  4. Algorithm 4: Donation Surgery - Surgeons perform organ retrieval
  5. Algorithm 5: Delivery - Transporters deliver organs safely
  6. Algorithm 6: Transplant Surgery - Surgeons perform transplantation

Technology Stack: Solidity ^0.8.0, Truffle, Ganache, Web3.js, OpenZeppelin AccessControl
